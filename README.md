# Web 工具站

基于 React + TypeScript + Tailwind CSS 构建的 Web 工具集合。

## 在线使用

访问 [https://t.19c.me](https://t.19c.me) 即可使用所有工具。

## 特性

- 🚀 快速响应，工具懒加载
- 🎨 简洁美观的界面设计
- 📱 支持桌面和移动端
- 🔒 纯前端实现，数据不上传服务器
- 🛠️ 持续更新的工具集合

## 现有工具

| 工具 | 描述 |
|------|------|
| **URL 编码/解码** | URL编码、解码及参数解析 |
| **Base64 编码/解码** | Base64编码解码，支持文件和URL Safe |
| **UUID 生成器** | 生成通用唯一识别码，支持多种版本 |
| **JWT 解码** | 解析和验证 JSON Web Token |
| **日期计算** | 计算 N 天以前或以后的日期 |
| **文本对比** | 对比两段文本的差异，支持行内高亮 |
| **文本搜索** | 在文本中搜索关键词并高亮显示 |
| **文件哈希校验** | 计算文件哈希值，校验文件完整性 |
| **屏保** | 浏览器桌面屏保，支持图片/视频/3D动画 |
| **Unix 时间戳** | Unix 时间戳与日期时间相互转换 |

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
